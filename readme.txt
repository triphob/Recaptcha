recaptcha
recaptcha เป็นการตรวจสอบผู้ใช้ก่อนเข้าถึงข้อมูล เพื่อเช็คว่าผู้ใช้ได้เข้ามาเองหรือเปล่า 
หรือใช้อย่างอื่นเพื่อเข้าถึงข้อมูล
#จุดประสงค์
1.เพื่อฝึกการเขียนโปรแกรม

2.เพื่อทำความเข้าใจในบทเรียนและนำมาประยุกต์ใช้
